# **Accessing and Updating Auto-Reply in a Shared Mailbox via OWA (Outlook Web Access)**

This guide is designed to help you access a Shared Mailbox and update its auto-reply settings using Outlook Web Access (OWA). Ensure you have the necessary delegation rights to the Shared Mailbox before following these steps.

## **Step-by-Step Instructions**

### **1. Access OWA (Outlook Web Access)**
- **Open Web Browser:** Launch your preferred web browser.
- **Navigate:** Go to the Outlook on the Web (OWA) URL at [https://portal.office.com/](https://portal.office.com/).

### **2. Log in with Your Credentials**
- **Enter Details:** Type your primary email address and password.
- **Sign In:** Click the **Sign in** button.

### **3. Access the Shared Mailbox**
- **Profile/Initials:** After logging into your personal mailbox, click on your profile picture or initials in the top-right corner.
- **Open Another Mailbox:** Select **Open another mailbox** from the dropdown menu.
- **Select Mailbox:** In the search box, enter the email address of the shared mailbox. Choose it from the list.
- **Open Mailbox:** Click **Open** to launch the shared mailbox in a new tab or window.

### **4. Navigate to Auto-Reply Settings**
- **Settings Icon:** In the shared mailbox, click the gear icon (⚙️) in the top-right corner.
- **Search Settings:** Type “automatic replies” in the "Search all settings" box and select **Automatic replies** when it appears.

### **5. Update the Auto-Reply**
- **Activate Auto-Replies:** Toggle the switch to enable **Automatic replies**.
- **Set Timing (Optional):** Specify a start and end time for the auto-reply if needed. Skip this step for a continuous auto-reply.
- **Compose Message:** Write your auto-reply message in the text box.
- **Outside Organization Replies (Optional):** Toggle and set a different auto-reply message for senders outside your organization, if required.

### **6. Save Changes**
- **Save:** Click the **Save** button at the top of the pane to apply your settings.

### **7. Exit the Shared Mailbox**
- **Close Tab/Window:** After setting up the auto-reply, you can close the shared mailbox's tab or window to revert to your personal mailbox.
