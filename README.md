# Managed Services KB
Internal knowledgebase used by the RCIT O365/M365 Team.

# Run Locally
- Install [Python](https://www.python.org/downloads/) 3.10 or higher.
- Clone this repo ([recommend with github cli](https://cli.github.com/)).
> *gh repo clone rivcoit/MKDocs*
> or 
> *git clone https://github.com/rivcoit/MKDocs.git*
- Install [mkdocs](https://www.mkdocs.org/) and dependencies.
> pip install -r requirements.txt
- Run MKDocs
> mkdocs serve
